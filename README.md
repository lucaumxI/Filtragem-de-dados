# Filtragem-de-dados

Esse repositório consiste no pré-processamento de dois dataset disponibilizados pelo Sinan sobre dengue e chikungunya que serão usados para treinamento de dois modelos distintos, o primeiro sendo uma triagem inicial se um paciente está com arbovirose ou não e o segundo para dizer se essa arbovirose é dengue ou chikungunya

O repositório está organizado em duas pastas (Modelo 1 e Modelos 2(W.I.P)) e dentro de cada pasta há alguns arquivos .py onde o tratamento do dataset é feito além de um .md especificando qual tratamento foi feito e porquê.
